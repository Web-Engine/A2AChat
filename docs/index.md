## 프로젝트 개요
A2A Chat은  

# A2A Chat Platform

## 프로젝트 목표
Google의 A2A (Agent-to-Agent) Protocol을 기반으로 구축된 멀티 에이전트 채팅 플랫폼입니다.
여러 AI 에이전트를 생성하고 관리하며, 이들 간의 상호작용을 가능하게 합니다.

## 프로젝트 기능
- Agent 생성/관리
- MCP Server 설치/관리
- Agent에 MCP Server 및 Tools 권한 관리
- Agent 시간 기반 예약 실행
- API 요청을 통한 Agent 실행
- Agent간 관계 설정
- 채팅을 통한 Agent 상호작용

## 향후 추가 예정 기능
- Agent 실행 결과 관리
- Agent 상태 모니터링
- Agent 설정 관리
- Agent 그룹 관리
